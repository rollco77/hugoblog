# 코드 포함한 게시글 작성

## 코드를 포함한 게시글을 마크다운문법을 통해 작성해보자

예) html 문법을 markdown 문법을 사용하여 게시
```html
<a href="rollco7.github.io" target="_blank"> Rollco7 Blog</a>
```

위의 코드는 다음과 같이 작성할 수 있다.

```
```html
<a href="rollco7.github.io" target="_blank"> Rollco7 Blog</a>
````
```
